## Начало работы

Стянуть проект:

```sh
$ git clone https://github.com/Vostafi69/ZHKKH.git
```

Перейти в директорию проекта:

```sh
$ cd ZHKKH
```

Установить зависимости

```sh
$ npm i
```

Запустить проект

```sh
$ npm run dev
```
