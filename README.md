## Начало работы

Стянуть проект:

```sh
$ git clone https://github.com/Vostafi69/ZHKKH.git
```

Перейти в директорию проекта:

```sh
$ cd ZHKKH
```

Установить зависимости и запустить проект

```sh
$ npm i
$ npm run dev
```
